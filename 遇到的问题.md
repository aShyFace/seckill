# reverseRangeByScoreWithScores返回Tuple，然后类型转换报错的问题
出错代码
```java
// 实际上Tuple中值的类型是Integer，但是用Long接收也不会报错
Set<ZSetOperations.TypedTuple<Long>> feedDataSet = redisTamplate.reverseRangeByScoreWithScores(x1, x2, x3, x4, x5);

for (ZSetOperations.TypedTuple<Long> tuple: feedDataSet) {
    // 等你把它当Long取出来的时候，就报“Integer cannot cast to Long”的错
    // 个人猜测是因为 你直接当Long取出来作了一个类型转换，tuple类型又不可变，所以报错了。只有get一个新的数据拷贝，在那个拷贝数据上转换才行
    Long value = tuple.getValue();
}
```

修改之后的代码
```java
// 只能用Integer接收，否则后面类型转化会报错
Set<ZSetOperations.TypedTuple<Integer>> feedDataSet = redisTamplate.reverseRangeByScoreWithScores(x1, x2, x3, x4, x5);

for (ZSetOperations.TypedTuple<Integer> tuple: feedDataSet) {
    Long value = tuple.getValue();
}
```
